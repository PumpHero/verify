How do we pick a winner?

We use a random generated signed seed from random.org together with the open-source random-seed package.

With the random-seed package we get the possibility to generate recreatable results.

What is recreatable result? It means that if you insert the same input, you should always recieve the same output. We give you proof of how we achieved the result, this is called provably fair.

Coin selection logic? Once again, the random-seed package will use the seed to give us a number within a range. This range is based on how many coins we used during our pump.

If we used 10 coins, the range will be from 0 to 9. If the generated number is 5, the 6th coin will be the winner. The first number of the range is 0 because we use JavaScript arrays.

Code example
